# Product Requirements

## MVP Scope
- Authentication (email and phone baseline)
- Provider discovery and filtering
- Appointment lifecycle
- Health records management
- Symptom tracking and triage history
- Offline-first queue with sync visibility

## User Roles
- Patient
- Provider
- Admin

## KPI Targets
- Onboarding completion >= 70%
- Appointment completion >= 60%
- Sync success >= 99%
- Crash free sessions >= 99.5%
