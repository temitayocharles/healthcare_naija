# MVP Acceptance Criteria

## Auth
- User can sign in and sign out.
- Session restores on app reopen.

## Providers
- User can list providers and filter by type/state.

## Appointments
- User can create, reschedule, and cancel appointments.

## Health Records
- User can create and view health records metadata.

## Offline Sync
- Offline operations queue and replay on reconnect.
- Pending queue badge updates in UI.
