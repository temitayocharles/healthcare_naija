## Production Media Policy

1. Use healthcare-safe imagery only (no unrelated travel/transport visuals).
2. Prefer SVG illustrations for deterministic UI consistency.
3. Run legal/licensing checks for all external photos before commit.
4. Keep PHI out of all media assets.
